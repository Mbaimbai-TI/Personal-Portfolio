# 🚀 Personal Web Portfolio: Mbaimbai Takalani Ignitius

Welcome to my personal portfolio. This project was developed to showcase my academic background, technical skills, and passion for bridging the gap between **Mathematics and Computer Science**.

---

### **📌 Project Overview**
This portfolio is a responsive, single-page web application designed following core **Human-Computer Interaction (HCI)** principles. It features a modern "Cyber-Tech" aesthetic, interactive JavaScript elements, and real-time data integration via the GitHub API.

### **🛠️ Sections Included:**
1.  **Homepage / Introduction:** A dynamic welcome message with a real-time typing effect.
2.  **About Me:** Detailed academic background, skills, and professional career goals.
3.  **Projects:** Interactive showcase of my latest work, including a Java Calculator, Future Value Tool, and the Cyber-Cipher.
4.  **Contact:** Direct links to my professional profiles and email.

---

### **⚙️ Technical Stack & Tools:**
*   **HTML5:** Structured with semantic elements for accessibility and SEO.
*   **CSS3 (Custom Variables & Flexbox):** Responsive design with a unique "Cyber" theme and backdrop-blur effects.
*   **JavaScript (ES6+):** Implementation of the typing effect, particle background, and GitHub API integration.
*   **GitHub API:** Dynamically fetching and displaying project data.
*   **HCI Principles:** Applied visibility of system status, aesthetic and minimalist design, and user control.
